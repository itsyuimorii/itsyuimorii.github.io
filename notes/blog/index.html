# 🚀 Project Introduction

This project is a project I studied before, a more complete blog system, with common blog-related functions, the original intention is to solve the daily learning record and summary.

The blog system probably contains the following functions:

  • user login and registration
  • system rights management
  • publish and edit blogs, blog access, blog deletion, blog comments
  • personal information page display
  • Personal information modification
  • home page information display such as popular tags, popular users, popular blogs, newly published blogs and all blogs list, and will do the statistics of reading, commenting and likes.

It also supports full-text search, so you can quickly find the content you want. Of course, it also provides a backend management system for administrators to manage and maintain the blog system, such as user management, blog management, etc.
